## markdown

- atom markdown preview

  - toggle preview: **ctrl+shit+M**
  - Github style: preview for Github


- [markdown-table-editor](https://atom.io/packages/markdown-table-editor)


- [tidy-markdown](https://atom.io/packages/tidy-markdown)
